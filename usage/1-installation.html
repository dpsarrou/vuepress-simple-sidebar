# Installation

  1. To install the plugin simply run

     npm install vuepress-simple-sidebar

  2. In your .vuepress/config.js add the plugin to the included plugins as

     plugins: [
     ...
     'vuepress-simple-sidebar',
 ]

  3. Thats it!
